# charity-water-game

A polished browser game inspired by charity: water's mission.

## Play Concept

In **Clean Water Rush**, players collect clean water drops and avoid pollution spills before time runs out.

## Features

- Difficulty modes: Easy, Normal, Hard
- Timed runs with different goals and pacing per difficulty
- Dynamic DOM interactions: drops and hazards are spawned and removed as players interact
- Milestone messages at key score thresholds
- Sound effects for collect, penalty, and win events
- Mission-aligned footer links to charity: water and donation page
- On-brand inspired typography and color direction

## Local Run

1. Open [index.html](index.html) in your browser.
2. Choose a difficulty mode.
3. Click **Start Run** and play.

## Deploy on GitHub Pages

1. Push this repository to GitHub.
2. Go to **Settings > Pages**.
3. Under **Build and deployment**, set source to **Deploy from a branch**.
4. Select branch **main** and folder **/(root)**.
5. Save and wait for deployment.
6. Your game URL will be:
	`https://<your-username>.github.io/charity-water-game`
